Project Description

This project is a custom implementation of the popular JavaScript utility library Lodash. It includes several commonly used functions recreated from scratch to demonstrate my understanding of JavaScript fundamentals.

Technologies Used

  • JavaScript
  • Node.js (for testing)

Features

  • chunk: Splits an array into groups of a specified size.
  • clamp: Clamps a number within a specified range.
  • drop: Creates a slice of an array with a specified number of elements dropped from the beginning.
  • dropWhile: Drops elements from an array while a condition is met.
  • findKey: Finds the key of the first element that satisfies a condition.
  • has: Checks if an object contains a specified key.
  • inRange: Checks if a number is within a specified range.
  • invert: Inverts the keys and values of an object.
  • pad: Pads a string with spaces to a specified length.
  • words: Splits a string into an array of words.
