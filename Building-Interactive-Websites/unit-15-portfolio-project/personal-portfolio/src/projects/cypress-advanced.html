Project Description

This project is part of the advanced module from the Talking About Testing (TaT) school. It demonstrates advanced testing techniques using Cypress, including end-to-end testing, custom commands, and advanced assertions.

Technologies Used

  • Cypress
  • JavaScript
  • Node.js

Features

  • End-to-end testing for web applications.
  • Custom Cypress commands for reusable test logic.
  • Advanced assertions for robust test validation.
  • Integration with CI/CD pipelines.
